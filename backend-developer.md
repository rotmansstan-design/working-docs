# Backend Developer — Go / PostgreSQL

**Формат:** Полная занятость, удалёнка
**Уровень:** Middle+ / Senior

## О проекте

Продукт в сфере онлайн-развлечений с крипто-механиками. Ищем бэкенд-инженера для построения надёжного, безопасного и масштабируемого API с нуля.

## Задачи

- Проектирование и разработка backend-сервисов на **Go**
- Работа с **PostgreSQL**: схема БД, индексы, оптимизация запросов
- Разработка REST API, интеграция с внешними сервисами и платёжными провайдерами
- Реализация финансовой логики: транзакции, балансы, вывод средств
- Настройка безопасности: аутентификация, антифрод, rate limiting
- Участие в архитектурных решениях, code review

## Требования

- 4+ лет опыта в backend-разработке, из них 2+ на **Go**
- Глубокое понимание PostgreSQL: транзакции, индексы, explain analyze
- Опыт проектирования REST API (OpenAPI / Swagger)
- Уверенное знание безопасности: JWT, refresh tokens, httpOnly cookies, CSRF, CSP
- Опыт интеграции с платёжными системами или PSP
- Умение писать чистый, тестируемый код

## Плюсом будет

- Опыт с крипто-платёжными интеграциями (blockchain, crypto withdrawals)
- Понимание принципов **Provably Fair** алгоритмов
- Опыт построения антифрод-систем, rate limiting, защиты от бонус-фарминга
- Опыт с Docker, Kubernetes
- Опыт работы с очередями (Kafka / RabbitMQ / NATS) и webhook'ами
- Практика в gambling / fintech-продуктах

## Условия

- Полная занятость, полностью удалённо
- Конкурентная зарплата — обсуждается индивидуально
- Прямое участие в формировании технической культуры команды
