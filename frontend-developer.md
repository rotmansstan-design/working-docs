# Frontend Developer — React / Next.js

**Формат:** Полная занятость, удалёнка
**Уровень:** Middle+ / Senior

## О проекте

Продукт в сфере онлайн-развлечений с крипто-механиками. Строим команду с нуля — ищем инженера, который будет влиять на архитектуру с первого дня.

## Задачи

- Разработка и поддержка клиентской части на Next.js (App Router / RSC)
- Проектирование компонентной архитектуры, переиспользуемых UI-компонентов
- Реализация платёжных флоу и интеграция с крипто-кошельками
- Разработка визуальных механик с анимациями
- Интеграция с REST API
- Участие в code review, формирование стандартов разработки

## Требования

- 4+ лет опыта во фронтенд-разработке
- Уверенное владение **React** и **Next.js** (App Router / SSR / SSG / RSC)
- TypeScript — как само собой разумеющееся
- Опыт работы с **Zustand** или аналогами (Redux Toolkit, Jotai)
- Понимание безопасности на стороне клиента: httpOnly cookies, CSP, XSS-prevention
- Умение читать дизайн в Figma и реализовывать pixel-perfect вёрстку
- Понимание производительности: Core Web Vitals, lazy loading, bundle optimization

## Плюсом будет

- Опыт интеграции Web3 / крипто-кошельков (MetaMask, WalletConnect и др.)
- Опыт с анимациями: GSAP, Framer Motion, CSS-анимации
- Знакомство с платёжными флоу и PSP-интеграциями
- Опыт в gaming / gambling / fintech-продуктах
- Практика настройки CI/CD для фронтенда

## Условия

- Полная занятость, полностью удалённо
- Конкурентная зарплата — обсуждается индивидуально
- Небольшая команда, прямое влияние на продукт
